# gerenciador_de_tarefas
Atualização para nova execução do CI
## Gestão de Mudanças

Durante o desenvolvimento do projeto, foi identificada a necessidade de adicionar
um campo de prioridade nas tarefas cadastradas.

Essa alteração teve como objetivo melhorar a organização do fluxo de trabalho,
permitindo que a equipe identifique tarefas mais importantes e urgentes.

A mudança foi implementada no sistema através da inclusão do atributo prioridade,
atualização do banco de dados e adaptação dos testes automatizados.

O Kanban do GitHub Projects também foi atualizado para refletir essa evolução
do projeto, mantendo o controle das atividades conforme a metodologia ágil.
